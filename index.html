Discord Simple Translate - Instalação

How to Install

  1. Install the Tampermonkey extension for your browser.
  2. Go to the Tampermonkey settings and enable "Developer Mode" if required.
  3. Click the button below to install the translation script for Discord.

Como Instalar

  1. Instale a extensão Tampermonkey no seu navegador.
  2. Vá para as configurações do Tampermonkey e ative o "Modo Desenvolvedor" se necessário.
  3. Clique no botão abaixo para instalar o script de tradução para Discord.

Cómo Instalar

  1. Instale la extensión Tampermonkey en su navegador.
  2. Vaya a la configuración de Tampermonkey y habilite el "Modo de Desarrollador" si es necesario.
  3. Haga clic en el botón a continuación para instalar el script de traducción para Discord.

Comment Installer

Как установить

  1. Установите расширение Tampermonkey для вашего браузера.
  2. Перейдите в настройки Tampermonkey и включите "Режим разработчика", если необходимо.
  3. Нажмите на кнопку ниже, чтобы установить скрипт перевода для Discord.
